# ML Performance Comparison — Heart Disease Dataset

## Objective
Compare multiple ML algorithms and identify the best performing one.

## Dataset
Heart Disease Dataset (UCI / Kaggle)

## Algorithms Used
- Logistic Regression
- K-Nearest Neighbors (KNN)
- Decision Tree

## Results
| Algorithm | Accuracy | F1 Score |
|---|---|---|
| Logistic Regression | ~85.25% | ~0.86 |
| KNN | ~90.16% | ~0.86 |
| Decision Tree | ~75.41% | ~0.74 |

## How to Run
Open `Major_Project.ipynb` in Google Colab and run chunks in order.
